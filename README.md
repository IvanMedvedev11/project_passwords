# Суть проекта
### Python-приложение которое:
- Хранит пользовательские пароли
# Проблема
В наше время необходимо помнить большое количество паролей, при этом, другие пользователи не должны о них узнать. Эту проблему решает надежный менеджер паролей, однако существующий аналог предназначен для ОС Windows
# 🎯Цель проекта
Цель: создание менеджера паролей для Linux на Python
# Задачи
- Создать виртуальную машину на базе Linux
- Разработать надежное приложение для хранения паролей на Python
- Протестировать приложение
# Целевая аудитория
- Пользователи Linux, которым важна безопасность своих паролей.
# Основные функции
- Создание, изменение, удаление и вывод паролей
- Выполнение SQL-запросов для получения данных из базы
- Выполнение SQL-запросов для изменения базы данных
# Как пользоваться
1. Клонировать репозиторий с помощью git clone
2. Установить зависимости с помощью pip install -r requirements.txt
3. Запустить скрипт с помощью ./run.sh
